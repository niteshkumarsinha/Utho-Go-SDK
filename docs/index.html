Installation

The project provides both a library for developers and a tool for sysadmins.

Go SDK

go get github.com/niteshkumarsinha/utho-sdk-go

Utho CLI

Build the binary from the root of the repository:

go build -o utho ./cmd/utho/main.go
./utho --help

Authentication & Configuration

Authentication is handled via a Bearer token (API Key).

1. Environment Variable

Both SDK and CLI automatically look for UTHO_APIKEY:

export UTHO_APIKEY="your-api-key"

2. Configuration File

The CLI can store your key in ~/.utho/config.json:

./utho configure

Programmatic Setup

// Uses default config (env or file)
client, _ := utho.NewClient("")

// Use explicit key
client, _ := utho.NewClient("explicit-key")

Utho CLI Usage

The CLI follows a standard utho pattern.

# Basic commands
./utho cloudserver list
./utho account info
./utho vpc list
./utho storage list

Check ./utho --help for a complete list of commands.

Cloud Server

Deploy and manage high-performance cloud instances with full lifecycle control.

Available Operations

  • List() - List all cloud servers
  • Get(id) - Get server details
  • Deploy(params) - Deploy new servers
  • Delete(id) - Destroy a server
  • PowerOn(id) - Start a server
  • PowerOff(id) - Stop a server
  • HardReboot(id) - Reboot a server
  • ResetPassword(id) - Reset root password

SDK Usage

// List all servers
servers, err := client.CloudServer.List()

// Get specific server
server, err := client.CloudServer.Get("server-id")

// Deploy a new server
params := cloudserver.DeployParams{
    DCSlug:         "inmumbaizone2",
    PlanID:         "10045",
    BillingCycle:   "hourly",
    Auth:           "password",
    EnablePublicIP: "1",
    Image:          "ubuntu-20.04-x64",
    Cloud: []cloudserver.InstanceConfig{
        {Hostname: "my-server"},
    },
}
resp, err := client.CloudServer.Deploy(params)

// Power operations
err = client.CloudServer.PowerOff("server-id")
err = client.CloudServer.PowerOn("server-id")
err = client.CloudServer.HardReboot("server-id")

// Delete server
err = client.CloudServer.Delete("server-id")

CLI Commands

# List servers
utho cloudserver list

# Get server details
utho cloudserver get <server-id>

# Deploy server
utho cloudserver deploy --zone inmumbaizone2 --plan 10045 --image ubuntu-20.04-x64 --hostname my-server

# Power operations
utho cloudserver poweroff <server-id>
utho cloudserver poweron <server-id>
utho cloudserver reboot <server-id>

# Delete server
utho cloudserver delete <server-id>
