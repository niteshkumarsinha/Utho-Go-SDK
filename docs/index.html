Installation

The Utho Go SDK is compatible with Go 1.18 and later.

Go Modules

Install the SDK using the go get command:

go get github.com/niteshkumarsinha/utho-sdk-go

Import the SDK

To use the SDK in your project, import the base package:

import "github.com/niteshkumarsinha/utho-sdk-go"

Authentication

All requests to the Utho API must be authenticated using a Bearer token (API Key).

Obtaining an API Key

You can manage your API keys through the Utho Console or programmatically via the Security service.

Using the API Key

Pass your API key when initializing the client:

client, err := utho.NewClient("your-api-key")

For better security, we recommend using environment variables:

apiKey := os.Getenv("UTHO_API_KEY")
client, err := utho.NewClient(apiKey)

Cloud Server

Deploy and manage high-performance cloud instances.

List Instances

servers, err := client.CloudServer.List()
for _, s := range servers {
    fmt.Println(s.Hostname, s.IP)
}

Deploy Instance

params := cloudserver.DeployParams{
    DCSlug: "inmumbaizone2",
    PlanID: "10045",
    Image: "ubuntu-22.04",
    Cloud: []cloudserver.InstanceConfig{{Hostname: "web-01"}},
}
resp, err := client.CloudServer.Deploy(params)
